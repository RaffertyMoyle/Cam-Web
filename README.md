# Cam-Web

A simple real-time video chat website built with WebRTC and Socket.io.

## Features

- Join a video chat room via a link
- Webcam activation upon joining
- Real-time video and audio communication
- Multiple users can join the same room

## Installation

1. Clone the repository
2. Run `npm install` to install dependencies

## Usage

1. Start the server: `npm start`
2. Open your browser and go to `http://localhost:3000`
3. Click "Join Chat" to activate your webcam and join the video chat
4. Share the link with others to let them join

## Development

For development with auto-restart: `npm run dev`

## Technologies Used

- Node.js
- Express
- Socket.io
- WebRTC
